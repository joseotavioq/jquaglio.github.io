AJAX - Asynchronous Javascript And XML

Nov 21, 2007

  • JavaScript

O AJAX é utilizado para fazer requisições em background, sem que a página realize uma atualização.

Com o AJAX, as páginas ficaram mais rápidas de ser carregadas, pois não precisamos mais carregar a página toda para cada link clicado, diminuindo assim o download de informações desnecessárias (como: imagens, css, javascript, etc).

Abaixo, coloquei um exemplo de como fazer uma chamada assíncrona a uma página; Resgatar o conteúdo dela e jogar em um div.
